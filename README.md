# Android Checker

made by mianxed, aka defozys

## Features
- Full storage scan for modified/created files in the last 24 hours
- Keyword-based detection (cheats, mods, injectors)
- Installed apps scanner with keyword matching
- Real-time log search with match navigation

## Download
Latest APK → [Releases](../../releases/latest)

## Usage
1. Install APK
2. Grant "All files access" permission
3. Press `Check`
